Tentang Saya

Butuh Produk Kreatif? Saya dapat membantu Anda!

Sebagai web developer dengan satu tahun pengalaman, saya telah terlibat dalam pengembangan dan pemeliharaan berbagai proyek web. Saya memiliki kemampuan untuk bekerja dengan HTML, CSS, dan JavaScript, serta pengalaman dengan framework seperti React dan Vue.js. Selain itu, saya juga menguasai Adobe Animate dan Figma, yang membantu saya dalam membuat animasi web dan desain UI/UX yang interaktif. Pengalaman saya mencakup bekerja dalam tim dan berkolaborasi dengan desainer dan pengembang lain untuk menciptakan solusi web yang efisien dan efektif.
